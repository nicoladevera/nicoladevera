# Hey, I'm Nicola 👋

I'm a Product Manager using AI coding agents to build personal software. I bring the problem definition, product judgment, and the standard for what "good" actually looks like; the AI writes the code. The work is in knowing what to build, how to evaluate whether it's actually good, and when to push for something better.

I believe the best products come from personal experience with the problem. I build things I actually need, which keeps me honest about what I'm solving. And I care about the difference between something made well and something made with care — it's detectable, even when you can't name the mechanism.
