## Projects

### [Code Guro](https://github.com/nicoladevera/code-guro) - Codebase Learning Tool

**Problem:** As a PM using AI coding agents, I could build functional prototypes but struggled to truly understand the generated code. Many PMs face this same gap.

**Solution:** CLI tool that analyzes codebases and generates beginner-friendly learning documentation. Published to PyPI with dual-format output (HTML + markdown), framework-specific context, and interactive Q&A mode.

**Tech:** Python, Anthropic Claude API, Click, Rich, tiktoken

---

### [PARfolio](https://github.com/nicoladevera/parfolio) - AI-Powered Interview Prep

**Problem:** Mid-career professionals struggle to recall and articulate their work experiences during interviews. Career stories stay messy and unstructured in our heads.

**Solution:** Voice-first app that listens to rambling work stories and converts them into structured PAR (Problem-Action-Result) narratives. Built complete AI orchestration pipeline from audio transcription to behavioral competency tagging.

**Tech:** Flutter (Web), FastAPI, Google Gemini 2.0 Flash, Google Cloud Speech-to-Text, ChromaDB, Firebase

---

### [Dossier Lab](https://github.com/nicoladevera/dossier-lab) - Personal Intelligence Tool

**Problem:** I read a lot — articles, papers, videos — but couldn't actually use what I'd read. Content lived in separate places with no way to search across it or connect ideas.

**Solution:** Ingests articles, YouTube videos, PDFs, and documents into a searchable knowledge base. Ask natural language questions and get streaming answers with source-level citations. Built an evaluation dashboard to track retrieval accuracy and response quality — not just whether it runs, but whether it's actually good.

**Tech:** Next.js, PostgreSQL + pgvector, Prisma, Tailwind CSS v4, OpenAI/Anthropic

---

### [Thinking Stack](https://github.com/nicoladevera/thinking-stack) - Structured Reasoning Tools

**Problem:** Most AI tools default to open-ended conversation. I wanted reusable reasoning tools that produce consistent, structured deliverables for high-stakes decisions.

**Solution:** Five Claude Code slash commands covering the full arc from belief to action: failure analysis, consequence mapping, belief audits, multi-agent debate, and executive summaries. Commands compose into two full reasoning chains.

**Tech:** Claude Code (slash commands), Markdown
